ER-079 Avoid casting primitive data types to lower precision

Severity2
Enabledyes
Waivable
Violation 
public long deliver(final int amount) {
	return stock - (long) amount;
}
Fix 
private long stock = 0L;
public long deliver(final long amount) {
	return stock - amount;
}
