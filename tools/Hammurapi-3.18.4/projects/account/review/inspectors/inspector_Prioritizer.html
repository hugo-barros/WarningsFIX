Prioritizer Generates priority metric. This metric is calculated as 100/((ln(clients+1)+1)*violationLevel+1) where clients is a number of classes which depend on classes in this file and violationLevel is a weighed sum of violations in the file. In a nutshell the worse the code in the file the lower priority metric value and as such the higher its position in the list.
